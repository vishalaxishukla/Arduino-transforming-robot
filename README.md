# Arduino Transformer Robot

## Overview
Built a transforming robot using Arduino capable of switching between car and humanoid modes. Focused on motor coordination and embedded control logic.

## Tech Stack
- Arduino
- Embedded C
- Servo Motors
- DC Motors

## Features
- Dual-mode transformation (car ↔ humanoid)
- Servo-based joint movement
- DC motor mobility system
- Smooth transition control logic

## My Contribution
- Hardware design and motor integration
- Implemented transformation sequence logic
- Coordinated servo and DC motor operations

## Learnings

- Motor synchronization challenges
- Embedded system control flow
